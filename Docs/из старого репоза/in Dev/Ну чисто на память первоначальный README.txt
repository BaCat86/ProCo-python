# Вкратце о проекте
Этот проект - консольное приложение (TUI), предназначенное для создания чего-то вроде персональной карточки, досье, личного дела человека. Туда могут помещаться различнейшие данные, такие как ФИО, номер телефона, адрес, аккаунт в соц. сетях и т.п. Инструмент должен быть лёгким, удобным и функциональным в использовании. Должны быть макеты "карточек", которые можно как либо редактировать, удаляя и добавляя поля. Так же хотелось бы иметь быстрый экспорт в разные форматы, такие как markdown, html, pdf и т.д. Должна быть возможность быстро приступить к работе, добавив тот или инной флаг при запуске программы (к примеру мгновенное открытие новой "карты", переход к уже существующей и т.п), а так же возможность группировать и ссылаться на различные карты. Касательно кода, то это будет opensourse проект с лицензией MIT, написанный на python.
